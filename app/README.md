# tigro Android App

The tigro android app is a platform that is intended to highlight core features of the tigro protocol. Specifically, the tigro protocol allows users to establish in-person connections via QR code. Once a connection is made, users are able to share annotations with people in their network.

# Building from Source

1. Download and install [Android Studio](https://developer.android.com/studio)
2. Launch Android Studio and select **Get from Version Control** from the Welcome Screen.
3. For URL, use https://github.com/tigro-brown/tigro-app.git
4. Wait for gradle to sync
5. Build proto files [see here](app/src/main/proto/PROTO-README.md)
